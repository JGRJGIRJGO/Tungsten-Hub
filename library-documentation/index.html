Introduction

Lyra UI Library is a premium, high-performance UI library designed specifically for Roblox script developers. Inspired by Microsoft's Fluent Design Guidelines and Rayfield's sidebar architecture, Lyra UI Library provides a gorgeous, responsive, and incredibly smooth user interface for exploit utilities, cheat hubs, and admin systems.

Core Philosophy

Most Roblox libraries suffer from performance drops, blocky layouts, or outdated color palettes. Lyra UI Library addresses these issues with custom rendering utilities, built-in layout scales, dynamic themes, and a clean, responsive navigation layout.

Feature Checklist

✔ Smooth sliding animations for tabs and window transitions

✔ Comprehensive elements: Buttons, Toggles, Sliders, Dropdowns, Textboxes, and Labels

✔ Multi-theme system: Switch seamlessly between Lyra, BloodMoon, Nebula, and Emerald palettes

✔ Dynamic Notification Engine built-in

✔ Optional FNV-1a HWID-Locked Key System with automated caching

Getting Started

To use Lyra UI Library in your Roblox script, import the library using the global loadstring below. This fetches the latest production build from the official GitHub repository.

Loader Script

Copy and paste this script directly into your Roblox executor:

local Lyra = loadstring(game:HttpGet("https://raw.githubusercontent.com/JGRJGIRJGO/Lyra-Hub/main/UniversalV2.lua"))()

Windows & Tabs

Every hub begins with a Window. Windows hold the global configurations, title, subtitle, optional key verification gate, and tabs.

Creating a Window Lyra:CreateWindow()

Use the CreateWindow method to configure the UI window properties. You can pass either a simple title string or a complete configuration table.

-- Standard configuration table
local Window = Lyra:CreateWindow({
    Name = "Lyra UI Library",
    Subtitle = "Universal Cheats"
})

Window Properties Table:

Property Type Description Default
Name string Main title displayed at the top-left of the window header. "Lyra UI Library"
Subtitle string Smaller caption displayed directly below the main title. "Universal V2"
KeySettings table Configuration block to enable the HWID gate locks (see Key System Gate tab). nil

Creating Tabs Window:CreateTab()

Tabs let you separate your components into categorized columns. Active tabs feature a left accent bar indicating selected state.

-- Create several tabs on the sidebar navigation
local MainTab = Window:CreateTab("Main")

Parameters: title (string) - Name of the tab shown in the sidebar menu.

Returns: A Tab Object that exposes element-creation methods (CreateButton, CreateToggle, etc.).

Global Window Actions

Once you have a window, you can run global management functions on it:

  • Window:Notify(title, description, duration) - Triggers a glassmorphic notification banner featuring a shrinking neon timer progress bar, manual dismiss close button, and automatic vertical stacking support.
  • Window:SetToggleKey(Enum.KeyCode) - Binds a custom keyboard hotkey to minimize/maximize the window interface dynamically (Default: RightShift).
  • Window:Destroy() - Cleanly removes the UI elements from CoreGui/PlayerGui.

UI Elements

UI Elements are interactive controls added to tabs. All elements scale automatically to fit content cleanly.

Buttons Tab:CreateButton()

Executes a block of code immediately upon user click. Features a clean card style and dynamic outline highlight transitions.

local MyButton = MainTab:CreateButton("Click Me!", function()
    print("Button clicked!")
end)

Parameters:

  • text (string) - Display text.
  • callback (function) - Executed on click.

Returns: A table containing UpdateButtonText(newText) to dynamically change the text of the button.

Toggles Tab:CreateToggle()

A compact switch control that maintains an ON/OFF state. Sends a boolean state to the callback. Implements a fluid sliding knob switch animation.

local MyToggle = MainTab:CreateToggle("Enable Autofarm", false, function(state)
    print("Autofarm active:", state)
end)

Parameters:

  • toggleText (string) - Label text.
  • default (boolean) - Initial state (true = ON, false = OFF).
  • callback (function) - Executed when state shifts, receiving the new boolean state.

Returns: A table containing SetToggle(boolean) to programmatically trigger toggling.

Sliders Tab:CreateSlider()

Lets users slide across a range of numbers. Perfect for walkspeed, jump power, or range sliders. Includes a sliding pointer handle that moves smoothly alongside the progress bar.

local MySlider = MovementTab:CreateSlider("WalkSpeed", 16, 200, 16, function(value)
    game:GetService("Players").LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Parameters:

  • sliderText (string) - Label text.
  • min (number) - Smallest allowed value.
  • max (number) - Largest allowed value.
  • default (number) - Starting value.
  • callback (function) - Executed during drag, receiving the current value.

Returns: A table containing SetValue(number) to force set a specific value programmatically.

Dropdowns Tab:CreateDropdown()

Collapsible lists allowing the user to pick a single string value from an array of choices. Smoothly expands downward with a clip-fade animation.

SettingsTab:CreateDropdown("Choose Region", {"US-East", "EU-West", "Asia"}, "US-East", function(selected)
    print("Region chosen:", selected)
end)

Parameters:

  • dropdownText (string) - Label text.
  • list (table) - Array of strings options.
  • default (string) - Initial selected option.
  • callback (function) - Executed on selection, receiving selected item text.

Textboxes Tab:CreateTextbox()

Accepts custom text input. Fires when the user presses Enter. Features clear-on-focus toggling and smooth focus indicators.

MovementTab:CreateTextbox("Custom Tag", "Type here...", function(text)
    print("User entered:", text)
end)

Parameters:

  • textboxText (string) - Label text.
  • placeholder (string) - Text to display when box is empty.
  • callback (function) - Executed on enter press, receiving the current text.

Labels Tab:CreateLabel()

Static headers or notice boxes used to organize elements or list info. Supports full multiline wrapping text blocks.

MainTab:CreateLabel("Version 2.1.0 Update")

Parameters: labelText (string) - Display content.

Themes & Styling

Lyra UI Library features a dynamic, run-time theme modifier. Themes update all labels, backgrounds, accent gradients, and borders in real-time without needing to redraw the UI.

Pre-made Theme Palettes

Lyra UI Library includes several premium themes preloaded:

  • Lyra.Themes.Lyra (Default Dark Acrylic Cyan)
  • Lyra.Themes.Nebula (Purple Galaxy Theme)
  • Lyra.Themes.BloodMoon (Dark Red Theme)
  • Lyra.Themes.Emerald (Deep Jade Green Theme)
  • Lyra.Themes.Midnight (Contrast Black Theme)

Applying a Theme Window:SetTheme()

Use the theme changer inside your code or bind it to a theme selector dropdown:

-- Dynamically load and apply a different palette
Window:SetTheme(Lyra.Themes.Nebula)

Key System Gate

Lyra UI Library includes an integrated SHA-256 HWID key gate. Developers can easily protect their scripts without having to purchase external hosting or database services.

Key System Configuration (Roblox Integration)

To enable the secure dynamic key verification, pass the KeySettings configuration block to CreateWindow:

local Window = Lyra:CreateWindow({
    Name = "Lyra UI Library",
    Subtitle = "Premium Edition",
    KeySettings = {
        Title = "Secure Key Gate",
        Subtitle = "Lyra Protection",
        Note = "Please pass the verification link to unlock the script.",
        SaveKey = true,                      -- Automatically saves key to workspace/
        Key = getDailyHWIDKey(),            -- Validating string (can also be an array/function)
        Url = "https://jgrjgirjgo.github.io/Lyra-Hub/?hwid=" .. getHashedHWID()
    }
})

KeySettings Properties Table:

Property Type Description
Title string Header title text displayed on the Key System window.
Subtitle string Smaller caption displayed directly below the Key System header.
Note string Instructions/description for the users explaining how to fetch the key.
SaveKey boolean If true, saves the successfully verified key locally in workspace/filename to bypass future loads within 24 hours.
Key string The exact Daily HWID Key that the player must enter to unlock the GUI.
Url string The redirection link (e.g. your Lyra-Hub page) loaded when users click "Get Key".

Detailed Daily HWID Verification Math (SHA-256)

The daily secure key is computed in Roblox using a secure cryptographical SHA-256 hash. This guarantees the highest level of security, preventing collision attacks or rainbow table lookups, and is fully synchronized with browser Web Crypto verification APIs.

How the hash is constructed step-by-step:

  1. Gather Inputs: The system collects the player's HWID (or Roblox UserId as fallback), hashes it once to get the secure Hashed HWID, and appends your custom Salt and the current UTC date formatted as DDMMYYYY.
  2. Concatenate: The string is formatted as: [HashedHWID]_[Salt]_[DDMMYYYY].
  3. Cryptographical Hashing: The concatenated string is processed through the 256-bit SHA-256 hashing function.
  4. Hex Formatting: The final output is represented as a 64-character uppercase hexadecimal string, prefixed with Lyra_.
-- Lua mathematical reproduction of the client-side SHA-256 key matching algorithm
local function generateDailyHWIDKey(hwid, salt)
    local hashedHwid = sha256(hwid):upper()
    local dateString = os.date("!%d%m%Y") -- UTC date formatting
    local input = hashedHwid .. "_" .. salt .. "_" .. dateString
    return "Lyra_" .. sha256(input):upper()
end

Complete Example

Here is a complete, copy-paste example showcasing initialization, multiple tabs, various interactive widgets, notifications, and theme modification controls.

Copy-Paste Script Template

Run this complete script in your Roblox executor to see all components working:

local Lyra = loadstring(game:HttpGet("https://raw.githubusercontent.com/JGRJGIRJGO/Lyra-Hub/main/UniversalV2.lua"))()

-- Create a window with title and subtitle
local Window = Lyra:CreateWindow({
    Name = "Lyra UI Library",
    Subtitle = "Combined Demo"
})

-- Create tabs
local MainTab = Window:CreateTab("Main")
local SettingsTab = Window:CreateTab("Settings")

-- Add label and notification button
MainTab:CreateLabel("Welcome to Lyra UI Library!")
MainTab:CreateButton("Send Notification", function()
    Window:Notify("Notification", "Lyra UI Library UI loaded!", 3)
end)

-- Add slider
MainTab:CreateSlider("WalkSpeed Booster", 16, 120, 16, function(val)
    game:GetService("Players").LocalPlayer.Character.Humanoid.WalkSpeed = val
end)

-- Add dynamic theme selector dropdown
SettingsTab:CreateDropdown("Change Theme", {"Lyra", "Nebula", "BloodMoon", "Emerald"}, "Lyra", function(theme)
    local themeTable = Lyra.Themes[theme]
    if themeTable then
        Window:SetTheme(themeTable)
    end
end)
