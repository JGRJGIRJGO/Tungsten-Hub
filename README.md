# Lyra UI Library

A clean, modern, and easy-to-use UI library designed to be simple for beginners while still being flexible enough for more advanced projects.

## Features

* ✨ Clean and modern interface
* 📖 Beginner-friendly API
* 🔑 Built-in SHA-256 key system
* ⚙️ Customizable key system
* 🔗 Currently supports Lootlink

You don't need much programming experience to get started—basic scripting knowledge is enough.
