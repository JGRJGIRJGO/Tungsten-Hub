Introduction

Tungsten Hub is a premium, high-performance UI library designed specifically for Roblox script developers. Inspired by Microsoft's Fluent Design Guidelines and Rayfield's sidebar architecture, Tungsten Hub provides a gorgeous, responsive, and incredibly smooth user interface for exploit utilities, cheat hubs, and admin systems.

Core Philosophy

Most Roblox libraries suffer from performance drops, blocky layouts, or outdated color palettes. Tungsten Hub addresses these issues with custom rendering utilities, built-in layout scales, dynamic themes, and a clean, responsive navigation layout.

Feature Checklist

✔ Smooth sliding animations for tabs and window transitions

✔ Comprehensive elements: Buttons, Toggles, Sliders, Dropdowns, Textboxes, and Labels

✔ Multi-theme system: Switch seamlessly between Tungsten, BloodMoon, Nebula, and Emerald palettes

✔ Dynamic Notification Engine built-in

✔ Optional FNV-1a HWID-Locked Key System with automated caching

Getting Started

To use Tungsten Hub in your Roblox script, import the library using the global loadstring below. This fetches the latest production build from the official GitHub repository.

Loader Script

Copy and paste this script directly into your Roblox executor:

local TungstenHub = loadstring(game:HttpGet("https://raw.githubusercontent.com/JGRJGIRJGO/Tungsten-Hub/main/UniversalV2.lua"))()

Windows & Tabs

Every hub begins with a Window. Windows hold the global configurations, title, subtitle, optional key verification gate, and tabs.

Creating a Window TungstenHub:CreateWindow()

Use the CreateWindow method to configure the UI window properties. You can pass either a simple title string or a complete configuration table.

-- Standard configuration table
local Window = TungstenHub:CreateWindow({
    Name = "Tungsten Hub",
    Subtitle = "Universal Cheats"
})

Creating Tabs Window:CreateTab()

Tabs let you separate your components into categorized columns. Active tabs feature a left accent bar indicating selected state.

-- Create several tabs on the sidebar navigation
local MainTab = Window:CreateTab("Main")
local MovementTab = Window:CreateTab("Movement")
local SettingsTab = Window:CreateTab("Settings")

UI Elements

UI Elements are interactive controls added to tabs. All elements scale automatically to fit content cleanly.

Buttons Tab:CreateButton()

Executes a block of code immediately upon user click.

MainTab:CreateButton("Click Me!", function()
    -- Action runs when clicked
    print("Button clicked!")
end)

Toggles Tab:CreateToggle()

A compact switch control that maintains an ON/OFF state. Sends a boolean state to the callback.

MainTab:CreateToggle("Enable Autofarm", false, function(state)
    -- state is true if turned on, false if off
    print("Autofarm active:", state)
end)

Sliders Tab:CreateSlider()

Lets users slide across a range of numbers. Perfect for walkspeed, jump power, or range sliders.

MovementTab:CreateSlider("WalkSpeed Booster", 16, 200, 16, function(value)
    -- value changes dynamically as user drags the slider handle
    game:GetService("Players").LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Dropdowns Tab:CreateDropdown()

Collapsible lists allowing the user to pick a single string value from an array of choices.

SettingsTab:CreateDropdown("Choose Region", {"US-East", "EU-West", "Asia-Pacific"}, "US-East", function(selected)
    print("Region chosen:", selected)
end)

Textboxes Tab:CreateTextbox()

Accepts custom text input. Fires when the user presses Enter.

MovementTab:CreateTextbox("Custom Tag", "Type here...", function(text)
    print("User entered:", text)
end)

Labels Tab:CreateLabel()

Static headers or notice boxes used to organize elements or list info.

MainTab:CreateLabel("Version 2.0.0 Update Changelog")

Themes & Styling

Tungsten Hub features a dynamic, run-time theme modifier. Themes update all labels, backgrounds, accent gradients, and borders in real-time without needing to redraw the UI.

Pre-made Theme Palettes

Tungsten Hub includes several premium themes preloaded:

  • TungstenHub.Themes.Tungsten (Default Dark Acrylic Cyan)
  • TungstenHub.Themes.Nebula (Purple Galaxy Theme)
  • TungstenHub.Themes.BloodMoon (Dark Red Theme)
  • TungstenHub.Themes.Emerald (Deep Jade Green Theme)
  • TungstenHub.Themes.Midnight (Contrast Black Theme)

Applying a Theme Window:SetTheme()

Use the theme changer inside your code or bind it to a theme selector dropdown:

-- Dynamically load and apply a different palette
Window:SetTheme(TungstenHub.Themes.Nebula)

Key System Gate

Tungsten Hub includes an integrated FNV-1a HWID key gate. Developers can easily protect their scripts without having to purchase external hosting or database services.

Configuration Example

To enable the secure dynamic key verification, pass the KeySettings configuration block to CreateWindow:

local Window = TungstenHub:CreateWindow({
    Name = "Tungsten Hub",
    Subtitle = "Premium Edition",
    KeySettings = {
        Title = "Secure Key Gate",
        Subtitle = "Tungsten Protection",
        Note = "Please pass the verification link to unlock the script.",
        SaveKey = true,                      -- Automatically saves key to workspace/
        Key = getDailyHWIDKey(),            -- Validating string (can also be an array/function)
        Url = "https://jgrjgirjgo.github.io/Tungsten-Hub/?hwid=" .. getHashedHWID()
    }
})

Complete Example

Here is a complete, copy-paste example showcasing initialization, multiple tabs, various interactive widgets, notifications, and theme modification controls.

Copy-Paste Script Template

Run this complete script in your Roblox executor to see all components working:

local TungstenHub = loadstring(game:HttpGet("https://raw.githubusercontent.com/JGRJGIRJGO/Tungsten-Hub/main/UniversalV2.lua"))()

-- Create a window with title and subtitle
local Window = TungstenHub:CreateWindow({
    Name = "Tungsten Hub",
    Subtitle = "Combined Demo"
})

-- Create tabs
local MainTab = Window:CreateTab("Main")
local SettingsTab = Window:CreateTab("Settings")

-- Add label and notification button
MainTab:CreateLabel("Welcome to Tungsten Hub!")
MainTab:CreateButton("Send Notification", function()
    Window:Notify("Notification", "Tungsten Hub UI loaded!", 3)
end)

-- Add slider
MainTab:CreateSlider("WalkSpeed Booster", 16, 120, 16, function(val)
    game:GetService("Players").LocalPlayer.Character.Humanoid.WalkSpeed = val
end)

-- Add dynamic theme selector dropdown
SettingsTab:CreateDropdown("Change Theme", {"Tungsten", "Nebula", "BloodMoon", "Emerald"}, "Tungsten", function(theme)
    local themeTable = TungstenHub.Themes[theme]
    if themeTable then
        Window:SetTheme(themeTable)
    end
end)
